## Blog App

-Un proyecto construido en Flask, un framework de Python el cual te permite agregar, editar y borrar tus posts
-El proyecto constituye tambien inicio de sesion, registro y reseteo de contraseña mediante correo electronico

# Tecnologias utilizadas

-Lenguaje: Python
-Framework: Flask
-Frontend: HTML5, CSS3, JavaScript(DOM)
-Entornos: venv (un entorno virtual ya integrado en python)
-BBDD: SQLite3

# Librerias

Consulta el archivo requirements.txt
